Build / Update the full-stack application:

"SPEED READING COMPETITION PLATFORM"

====================================================
TECH STACK
====================================================
- Node.js + Express backend
- PostgreSQL + Prisma ORM
- React (Vite) frontend
- Tailwind CSS
- JWT Authentication
- bcrypt password hashing
- All time logic controlled by server (UTC)
- Must run inside Replit
- Monorepo: /server and /client

STUDENT SYSTEM
====================================================

REGISTRATION
Fields:
name, surname, gender, birthdate, phone, city, country, email, password, category

- redirect to Student Dashboard

LOGIN
email + password

STUDENT DASHBOARD
Show:
- Profile info
- Category
- Affiliate code (copy + share)
- Referral points + list
- Competition info + countdown
- Assigned book
- Prize info
- Submission status

COMPETITION FLOW
Before start → countdown only
Start → reading unlocked
Finish reading → questions unlocked
Finish exam → store submission

"Submission received. Results will be released soon."

ADMIN DASHBOARD

- Reading Time
- Answer Time
- MCQ Correct / Wrong / Total
- Auto Score
- Manual Score
- Final Score
- Status
- View Details

Submission Details View (Answer Analysis Page)

Must show:

A) Summary
- Student profile
- Referral data
- Reading time
- Answer time
- MCQ statistics (correct / wrong / total)
- Score breakdown

B) MCQ Answers
For each MCQ:
- Question text
- Options A/B/C/D
- Student answer
- Correct answer
- Status badge: Correct / Wrong

C) Text Answers (Teacher Analysis)
For each TEXT question:
- Question text
- Student's full written answer
- Input field: Manual Score
- Save button
Teacher can:
- Read
- Analyze
- Score

All answers must be permanently stored.

====================================================
RANKING & RESULTS
====================================================

Final Score = MCQ Correct + Manual Score

Ranking rules:
1) Higher Final Score
2) Faster reading time
3) Faster answer time
4) Earlier submission

Admin can:
- Recalculate rankings
- Publish results per category
- Unpublish results

Students see leaderboard ONLY after publish.

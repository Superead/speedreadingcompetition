======================
STUDENT FEATURES
======================

REGISTRATION PAGE

Fields:
name
surname
gender
birthdate
phone
city
country
email
password
category

On registration:
- Validate registration window from admin settings
- Hash password
- Save user
- Auto-generate unique affiliateCode (8 chars)
- referralCode input is OPTIONAL (not required)
- If referralCode exists and matches an affiliateCode:
     set referrerId
     referrer.referralPoints += 1
- Auto login and redirect to Student Dashboard

LOGIN PAGE
email + password

STUDENT DASHBOARD

Show:
- Full Name
- Category
- Affiliate Code (copy button)
- Referral Points
- Referral List (users who joined with my code)
- Competition info for their category:
   registrationStartTime
   registrationEndTime
   competitionStartTime
   readingDurationMinutes
   answeringDurationMinutes
   Live countdown timer
- Assigned Book
- Prizes section
- Superead login button (external placeholder)

COMPETITION FLOW

Server is source of truth.

Before competitionStartTime:
- show countdown
- hide actions

When competition starts:
- show "Start Reading"

Start Reading:
- save readingStartAt (server time)
- unlock book
- start reading timer

Finish Reading:
- save readingEndAt
- calculate readingSeconds
- unlock questions
- start answering timer

Answering:
- Show MCQ and Text questions
- Save answers continuously

Finish Competition:
- save answerEndAt
- calculate auto score
- display final results

Auto-submit on timeout.
